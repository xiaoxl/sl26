2.1 Quick Overview

Linear regression is the basic supervised learning method for a quantitative response. It models the mean of the response as a linear function of the predictors:

\[ Y = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p + \varepsilon. \]

Equivalently, in matrix form,

\[ y = X\beta + \varepsilon. \]

2.2 The Ordinary Least Squares Estimator

The most common fitting method is ordinary least squares (OLS). It chooses the coefficient vector \(\beta\) that minimizes the residual sum of squares:

\[ \mathrm{RSS}(\beta) = \sum_{i=1}^n (y_i - \hat{y}_i)^2 = \lVert y - X\beta \rVert_2^2. \]

The OLS estimator is

\[ \hat{\beta}_{\mathrm{OLS}} = (X^\top X)^{-1}X^\top y, \]

provided that \(X^\top X\) is invertible.

Once the coefficients are estimated, the fitted value for observation \(i\) is

\[ \hat{y}_i = \hat{\beta}_0 + \hat{\beta}_1 x_{i1} + \cdots + \hat{\beta}_p x_{ip}. \]

2.3 Fitting Linear Regression in R

In R, linear regression is fitted with lm():

fit <- lm(y ~ x1 + x2, data = dat)
summary(fit)

The formula syntax is convenient:

  • y ~ x1 + x2 includes two predictors;
  • y ~ . includes all available predictors;
  • y ~ x1 * x2 includes main effects and the interaction;
  • y ~ x + I(x^2) adds a quadratic term.

2.4 Categorical Variables, Interactions, and Nonlinear Terms

Linear regression can handle more than purely numeric additive effects.

2.4.1 Categorical Predictors

Categorical predictors are represented through indicator variables. If a factor has \(k\) levels, the model usually includes \(k-1\) dummy variables.

Thus, linear regression can compare group means while adjusting for other predictors.

2.4.2 Interaction Terms

An interaction allows the effect of one predictor to depend on another predictor. For example,

\[ Y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \beta_3 x_1 x_2 + \varepsilon. \]

If \(\beta_3 \neq 0\), the effect of \(x_1\) depends on the value of \(x_2\).

2.4.3 Polynomial and Transformed Terms

We may also include nonlinear transformations such as:

  • \(x^2\),
  • \(\log(x)\),
  • \(\sqrt{x}\).

The model remains a linear model because it is still linear in the coefficients \(\beta\).

2.5 Interpreting the Model

Interpretation is one of the main strengths of linear regression.

2.5.1 Coefficients

The coefficient \(\beta_j\) represents the change in the mean response associated with a one-unit increase in \(X_j\), holding the other predictors fixed.

2.5.2 Main Effects With Interactions

When interactions are present, a main effect should not be interpreted in isolation. Its meaning depends on the interacting variables.

2.5.3 Confidence Intervals

A confidence interval gives a range of plausible values for a coefficient or for the mean response at a given predictor combination.

2.5.4 Prediction Intervals

A prediction interval is wider than a confidence interval because it must account for both:

  • uncertainty in the estimated mean;
  • random variation in a future observation.

2.6 Assessing Model Fit

Several tools are commonly used to assess a linear regression fit:

  • residual plots,
  • Q-Q plots,
  • leverage and influence diagnostics,
  • measures such as \(R^2\), adjusted \(R^2\), and residual standard error.

Residual plots help detect:

  • nonlinearity,
  • nonconstant variance,
  • outliers,
  • influential observations.

These diagnostics do not prove that a model is correct, but they help identify important problems.

2.7 Model Selection

In multiple regression, several competing predictor sets may be plausible. Model selection compares them using a criterion that balances fit and complexity.

2.7.1 AIC, BIC, and Mallows’ \(C_p\)

These criteria reward good fit but penalize model complexity.

  • AIC tends to prefer somewhat larger models.
  • BIC penalizes complexity more strongly and often prefers smaller models.
  • Mallows’ \(C_p\) compares a candidate model to a larger reference model.

These tools are useful for comparing models, but they should not replace subject-matter reasoning.

2.7.2 Best Subset Selection

Best subset selection compares all models of a given size and then chooses the best one according to a criterion such as AIC, BIC, adjusted \(R^2\), or \(C_p\).

It is conceptually simple, but it becomes computationally expensive when the number of predictors is large.

2.7.3 Stepwise Selection

Stepwise procedures add or remove variables sequentially rather than searching all possible models.

Common versions are:

  • forward selection,
  • backward elimination,
  • bidirectional stepwise selection.

These methods are faster than best subset selection, but they are greedy procedures and do not guarantee the globally best model.

ImportantCaution

If many models are tried and the final model is selected from the same data, p-values and confidence intervals from the final selected model should be interpreted cautiously.

2.8 Strengths and Limitations

Linear regression is often the first model to try because it is transparent and effective in many settings. However, it has limits.

Strengths:

  • simple interpretation;
  • fast computation;
  • useful baseline;
  • strong connection to statistical inference.

Limitations:

  • assumes a linear mean structure unless we add transformed terms;
  • can be unstable under multicollinearity;
  • may overfit when many predictors are included;
  • may perform poorly when relationships are strongly nonlinear.

These limitations motivate later methods such as GLMs, GAMs, ridge regression, and LASSO.

2.9.8 Generate ANOVA Tables: Type I, II, and III

Type I sums of squares depend on the order in which the terms enter the model. Type II and Type III tables are order-invariant under the usual model-coding assumptions, but they answer slightly different questions.

fit_anova = smf.ols(
    "mpg ~ horsepower + weight + acceleration + C(origin)",
    data=mpg,
).fit()

print("Type I ANOVA")
print(anova_lm(fit_anova, typ=1))

print("\nType II ANOVA")
print(anova_lm(fit_anova, typ=2))

print("\nType III ANOVA")
print(anova_lm(fit_anova, typ=3))

2.9.9 Generate a Nested ANOVA Table

Nested ANOVA compares a smaller model with a larger model that contains all of its terms plus additional predictors.

fit_nested_small = smf.ols("mpg ~ horsepower + weight", data=mpg).fit()
fit_nested_large = smf.ols(
    "mpg ~ horsepower + weight + acceleration + C(origin)",
    data=mpg,

2.9.14 Stepwise Selection

The following simple forward-selection routine uses AIC as the selection criterion. It is easy to implement, but it is still a greedy search rather than a global optimum.

def forward_selection(data, response, candidates):
    remaining = list(candidates)
    selected = []
    current_aic = np.inf
    history = []

    while remaining:
        scored = []
        for term in remaining:
            formula = f"{response} ~ {' + '.join(selected + [term])}"
            fit = smf.ols(formula, data=data).fit()
            scored.append((fit.aic, term, fit))
        scored.sort(key=lambda x: x[0])
        best_aic, best_term, best_fit = scored[0]
        if best_aic + 1e-8 < current_aic:
            selected.append(best_term)
            remaining.remove(best_term)
            current_aic = best_aic
            history.append((best_term, best_aic))
        else:
            break

    return selected, current_aic, pd.DataFrame(history, columns=["added_term", "aic"])


candidate_terms = [
    "horsepower",
    "weight",
    "acceleration",
    "C(origin)",
    "C(cylinders)",
]

selected_terms, selected_aic, stepwise_path = forward_selection(
    mpg, "mpg", candidate_terms
)

2.9.15 Best Subset Selection

Best subset selection fits every model of a given size and records the best one according to a chosen criterion.

def model_diagnostics(fit):
    y = fit.model.endog
    n = fit.nobs
    p = fit.df_model + 1
    sse = np.sum(fit.resid ** 2)
    mse = sse / fit.df_resid
    press = np.sum((fit.resid / (1 - fit.get_influence().hat_matrix_diag)) ** 2)
    return {
        "r2": fit.rsquared,
        "adj_r2": fit.rsquared_adj,
        "sse": sse,
        "mse": mse,
        "aic": fit.aic,
        "bic": fit.bic,
        "press": press,
        "n": n,
        "p": p,
    }


full_formula = "mpg ~ horsepower + weight + acceleration + C(origin) + C(cylinders)"
full_fit = smf.ols(full_formula, data=mpg).fit()
sigma2_full = np.sum(full_fit.resid ** 2) / full_fit.df_resid


def mallows_cp(fit, sigma2_full):
    sse = np.sum(fit.resid ** 2)
    p = fit.df_model + 1
    n = fit.nobs
    return sse / sigma2_full - (n - 2 * p)


subset_terms = [
    "horsepower",
    "weight",
    "acceleration",
    "C(origin)",
]

subset_rows = []
for k in range(1, len(subset_terms) + 1):
    for combo in itertools.combinations(subset_terms, k):
        formula = "mpg ~ " + " + ".join(combo)
        fit = smf.ols(formula, data=mpg).fit()
        row = {"terms": " + ".join(combo), "size": k}
        row.update(model_diagnostics(fit))
        row["cp"] = mallows_cp(fit, sigma2_full)
        subset_rows.append(row)
